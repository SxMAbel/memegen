## How to use Random Meme Generator

## Typescript

```ts
import { RandomMeme } from "memegen-techstack";

const initMeme = new RandomMeme();
const meme = await initMeme.randomMeme();
```

## Javascript

```js
const { RandomMeme } = require("memegen-techstack");

const initMeme = new RandomMeme();
const meme = await initMeme.randomMeme();
```

> **`meme` will return:**
- HDPreview
- Author
- NSFW
- Preview
- Spoiler
- Title
- URL
